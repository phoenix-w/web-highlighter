# [Extension] Web Highlighter
Highlight text on any web page. Highlights persist when you revisit.

## Features
- Select text and press Option + 1-6 to highlight in different colors
- Six default colors: red, orange, yellow, green, blue, and purple
- Highlights persist across browser sessions
- Click any highlight to change its color or remove it
- Add custom colors and remap keyboard shortcuts
- Adjust highlight opacity and border radius
- View and manage all saved highlights from the settings page
- Works on Chrome and Edge

## Privacy
All data is stored locally in your browser. Nothing is sent to any server.
